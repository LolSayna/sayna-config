
# File systems
## Windows
- fat32 - old windows standard, max file size 4GB
    - fat/fat12/fat16 - older variants
- exFAT - flash drives
- ntfs - windows

## Apple
- HFS - old apple 
- btrfs

## Linux
- ext4
    - ext/ext2/ext3 - older variants
- Btrfs
- zfs

## Special
- SquashFS - compressed read only
